[解决"修改文件名大小写"造成的git上传文件丢失](https://juejin.im/pin/5d31717e6fb9a07d872415b1)

法一

1. 让 git 区分大小写：

```sh
git config core.ignorecase false
```

2. 删除缓存：

```sh
git rm -r --cached
```

3. 剩余照常

```sh
git add .
git commit -m 'xxx'
git push xxx
```

法二

修改文件大小写，然后重新提交

```sh
a.txt A.txt
```
