[CSS :focus-visible](https://developer.mozilla.org/en-US/docs/Web/CSS/:focus-visible)

CSS 的 :focus-visible 伪类可以区分一个元素是通过鼠标点击聚焦的，还是通过 Tab 键索引聚焦的。

这样就能实现以下的交互效果——点击 `<button>`  时不会出现 outline，而用 Tab 键索引时出现 outline。
